# servidor-sisbanco
Sistema bancario simples

Esse é o protótipo de um sistema bancario simples, mas com regras de negocio e arquitetura bem definidas. 
Essa parte do projeto é somente o Servidor, ainda existe a implementação do Atendente do Caixa e o Caixa Eletrônico. 

Algumas tecnologias utilizadas:
- Java EE
- Java RMI
- PostgreSQL

Documentação:
dr-banco.dia ->  Diagrama de Entidade Relacionamento do Banco de Dados;
Engenharia do trabalho.odt ->  Documento com os Requisitos Funcionais, Casos de Uso e Regras de Negócio;
arquitetura-da-aplicação.dia ->  A forma como os componentes da aplicação estão organizados;
